## 🏆 Featured Projects

### 🌟 [Diverxia Events Suite](https://github.com/asanmun1905/DiverxiaProject)
> A premium, interactive event management application designed for smooth organization and scheduling of competitive events.
* **Frontend Excellence:** Fully styled in modern dark mode using sleek Glassmorphism principles (`backdrop-filter: blur`), responsive navigation, and floating custom animations.
* **Interactive Mechanics:** Native HTML5 Drag & Drop sorting (`Sortable.js`) to seamlessly arrange events, bulk management actions, and visual status indicators.
* **State & Data:** Integrated with **Supabase** (PostgreSQL) for remote persistence and Netlify for static deployment.
* **Tech Stack:** `Vanilla JS` • `HTML5 / CSS3` • `Supabase` • `SortableJS` • `Netlify`

### 🎮 [Hopeless Despair (Survivor Game)](https://github.com/asanmun1905/Hopeless-despair)
> A fast-paced, 2D web-based survivor game featuring automated waves and complex bullet patterns.
* **Graphics & Rendering:** Engine developed with a custom Canvas-based loop in TypeScript. Renders multi-frame spritesheets (like idle/movement directions) with precise clipping to eliminate visual artifacts.
* **Gameplay Mechanics:** Multiple modes including survival waves (Oleadas) with auto-attacks, integrated merchant/shop, and a highly challenging *Bullet Hell* mode featuring spiral and curtain bullet patterns.
* **Polish & Effects:** Dynamic visual feedback using semi-transparent Area of Effect (AoE) glows mapped to entity attack ranges.
* **Tech Stack:** `TypeScript` • `HTML5 Canvas` • `Vite` • `CSS3`
